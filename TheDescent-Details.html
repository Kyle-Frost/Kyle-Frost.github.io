The Descent

Project information

  • Category: Unity
  • Genre: Roguelike Dungeon Crawler
  • Project date: May, 2022
  • Project URL: GitHub Repository

A 3-week Game Jam based on a given theme - "A horror game where you explore dungeons full of trolls and loot sweet gear."

This game jam features randomly generated dungeons with varying designed rooms, created so each run will feel unique in its design. Equipped with a bow and elemental arrows, you can take down enemies to spawn randomised loot with power tiers - making you more powerful as you push through each floor of the dungeon. At the end of each dungeon floor is a boss with unique attacks to provide an extra challenge.

  • Randomised Room/Dungeon Spawning
  • Randomised Tier-Based Loot System
